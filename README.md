![NODstrap](/img/logo.strap.png?raw=true "NODstrap")

Simple bootstrap for web projects of [NOD studios](http://nod.st)

#Installation:
```bash
git clone git@github.com:NOD-studios/NOD-strap.git
cd NOD-strap
bash bin/update.bash
```

#Requirements
- Git
- Composer
- Node.js
- npm
