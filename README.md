NODstrap is bootstrapper for web project conventions of [NOD studios](http://nod.st).

##Installation:
```bash
git clone git@github.com:NOD-studios/NOD-strap.git
cd NOD-strap
npm install && jspm update && gulp watch
```

#Requirements
- Git
- npm
- Node.js
- Composer
- PHP
- Apache Server
